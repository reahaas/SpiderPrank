# SpaiderPrank
SpiderPrank

### Game play:
1. Android game.
2. Runner on round world.

### Movement: 
1 Button with changing functionality and display:
a. move right
b. move left
c. Lay an egg
d. Shot

### Scoring:
1. surviving: time * number of live enemies.
2. Kill enemies.
3. Laying eggs.

### Eggs:
change from a passive object to enemy object after certain amount of time
